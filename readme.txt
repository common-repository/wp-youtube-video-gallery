YouTube video gallery plugin for showing YouTube channel videos on your website. 
== Description ==

Plese, you can test your channel here. [Test Your Channel](http://youtube-gallery.moondeveloper.com/)
If you want a full control and all functionalities then you can also buy the pro version from this link. [Buy Pro Version](http://moondeveloper.com/product/wp-youtube-video-gallery/)


If you want to show your channel videos on your WordPress website with a beautiful way then this plugin is a right choice. WP Youtube Video Gallery plugin will show your YouTube channel videos including: 
*	Channel Art(banner)
*	Channel Logo
*	Channel name
*	Chanel subscribe button
*	Video Likes, views and comments count
*	Video comments on the video



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on 'WP Youtube Video Gallery' option on Setting tab.



== Frequently Asked Questions ==

= Can I add my Channel  =

Yes, you can add your channel going to 'Setting -> YT Video Gallery -> API'.
